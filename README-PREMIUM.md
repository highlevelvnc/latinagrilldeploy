### 🚀 Como Usar

```bash
# 1. Instalar dependências
npm install

# 2. Baixar os assets do GitHub
# Certifique-se de que todos os arquivos listados acima
# estão na pasta /public do projeto

# 3. Rodar em desenvolvimento
npm run dev

# 4. Build para produção
npm run build
npm start
```

### 🎨 Características do Design

#### Tipografia
- **Títulos**: Playfair Display (serif elegante)
- **Corpo**: Inter (sans-serif moderna)
- **Tracking**: Espaçamento generoso para premium feel

#### Cores Mantidas
- **Dark**: #0a0a0a (backgrounds)
- **Red**: #DC2626 (brand principal)
- **Accent Orange**: #F97316 (destaques)
- **Accent Yellow**: #EAB308 (acentos quentes)
- **Light**: #F5F5F5 (textos)

#### Animações
- Fade in suaves
- Hover effects em imagens (scale 1.05-1.10)
- Parallax sutil em scrolls
- Transições de 300-700ms

#### Layout
- **Espaçamento**: 24-32 py (mobile) / 32-40 py (desktop)
- **Containers**: max-width consistentes
- **Grid**: CSS Grid moderno
- **Responsivo**: Mobile-first approach
